Ixion - How To Make The Logo

Russell lets us into his artistic secrets:

How To Make A MetalIxiLogo In Twelve Easy Steps

  1. Get and fire up the gimp :-) (www.gimp.org)
  2. In the main gimp control window select Xtns->Script-Fu -> Logos -> SOTA Chrome
  3. It'll pop up a "Script-Fu: Logos/SOTA Chrome" window. In this window set:
    Chrome Saturation-80(default)
    Chrome Lightness-47(default)
    Chrome Factor0.75(default)
    TextIXION(this bit's quite important :-) )
    Font Size300(or wotever you like: this makes a 888x362 image)
    FontROOSTHEAVY(you might have to STFW & install the font, and/or it could have a different name e.g. "Rooster Heavy")
    Environment Map.../beavis.jpg(default)
    Highlight BalanceRGB 0.83 0.37 0.00
    (muddy brown)    		(default)
    Chrome BalanceRGB 0 0 0
    (black)    		(default)
  4. Click OK and the gimp should create a metalic logo with border.
  5. Get rid of the shadow & background:
    1. Open the Layers, Channels & Paths window by right clicking in the image then selecting Layers -> Layers, Channels & paths
    2. Select the background layer in the Layers window by clicking on the word Backgound then delete this layer by either typing ^X or right click and select Delete Layer.
    3. Select the Drop Shadow layer in the Layers window by clicking on the word Drop Shadow then delete this layer by either typing ^X or right click and select Delete Layer.
  6. Right click in the Layers window, select Merge Visible Layers then click OK.
  7. Get rid of the border:
    1. In the image window, left click and drag around the word "IXION", i.e. inside the border.
    2. Right click in the image window and select Select -> Invert.
    3. Right click in the image window and select Edit -> Clear.
    4. (optionally) Right click in the image window and select Image -> Transforms -> Autocrop
  8. Save Image with right click then File -> Save As and choose wot you like.

Tara! One Metal Ixion Logo of a size and file format chosen by your good self.
