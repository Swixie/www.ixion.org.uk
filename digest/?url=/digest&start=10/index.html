Re: This weeks cheap desktop?

Posted by andy pugh <bodgesoc-at-gmail.com> on 2 May 2015 | 0 Comments

On 1 May 2015 at 19:40, Swixie <swixie-at-gmail.com> wrote:
> It seems to me that you're saying that contrary to Paul's statements Windows
> is actually more complicated, and more difficult to install than are Linux
> distributions.

I don't actually think that that is true. But then it's been a long
time since I installed a Windows.

My Mac had a serious problem last week, and I was amazed how easy the
recovery was.

A disk scan showed a problem with the file system that needed the
drive to be reformatted. I don't have any physical media for the last
3 updates, but I found the OS from before that, and was fairly sure I
had a backup. However that wasn't needed.

The Disk Utility told me to reboot and press CMD-R during reboot. I
did that and the machine booted from a recovery partition I didn't
know I had. In there were three apps. The first was disk utility,
which reformatted the main drive. Then OS restore knew what OS I was
on, ran off to the App store to fetch it, and reinstalled Yosemite.
And then it rebooted, found my Time Machine Backup and restored.

So, a complete drive reformat took 3 button presses and two passwords
(WiFi key and App Store login) to get me back to exactly where I was
before. I was pleasantly surprised.
